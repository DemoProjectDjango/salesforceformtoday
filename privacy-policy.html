Privacy Policy

Last Updated: January 2026

Overview

Salesforce Form Today is a Salesforce application that allows authorized Salesforce users to connect their Google account, select Google Forms owned by them, and synchronize form responses into Salesforce objects based on user-defined field mappings. This policy explains what data is accessed, how it is used, and how it is protected.

Data We Access

The application accesses Google data only after explicit user consent and only for the following purposes:

  • Google Forms metadata (Form name, Form ID)
  • Google Forms questions and structure
  • Google Forms responses submitted to the selected form
  • Basic Google Drive access limited to locating Forms owned by the user
The application does NOT access:
  • Gmail content
  • Google Contacts
  • Google Calendar
  • Google Photos
  • Any Google data unrelated to Google Forms

How We Use Data

The accessed data is used solely to:

  • Allow users to select their Google Forms
  • Map Google Form questions to Salesforce fields
  • Insert or update Salesforce records using submitted form responses
  • Prevent duplicate processing using unique Google response identifiers

Note: No data is used for advertising, analytics, profiling, or resale.

Data Storage

  • Google Form response data is stored only inside the user’s Salesforce organization.
  • No Google data is stored outside Salesforce.
  • No third-party databases or external storage systems are used.

Data Sharing

  • We do not share Google user data with any third party.
  • Data is visible only to authorized Salesforce users within the same Salesforce organization.
  • Visibility is governed by Salesforce’s standard object-level and field-level security rules.

User Control & Security

User Control: Users can disconnect their Google account, stop synchronization, delete records, or remove OAuth access at any time.

Security: OAuth 2.0 is used for authentication. Named Credentials are used for secure token storage in Salesforce. No tokens are stored outside Salesforce.
