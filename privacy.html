Privacy Policy

Last Updated: January 2026

Overview

Form Today is a Salesforce app that integrates Google Forms with Salesforce. It allows users to authenticate their Google accounts, select forms, and synchronize responses into Salesforce based on mappings defined by the user.

Data We Access

The app accesses Google data only with explicit user consent for the following:

  • Google Forms metadata (form name, form ID)
  • Google Forms questions and structure
  • Responses submitted to selected forms
  • Limited Google Drive access to locate user forms
Note: The app does not access Gmail, Google Contacts, Google Calendar, or Google Photos.

How We Use Data

The app uses the data exclusively for:

  • Mapping Google Form fields to Salesforce fields
  • Inserting or updating Salesforce records with form responses
  • Preventing duplicate data processing

Data Storage

Google Form data is stored only in the user’s Salesforce organization, and no data is stored on third-party servers or databases.

Data Sharing

We do not share data with third parties. Data is visible only to authorized users within the same Salesforce organization, following Salesforce's object-level and field-level security.

User Control

Users can:

  • Disconnect their Google account at any time
  • Stop synchronization whenever desired
  • Delete synced records via Salesforce automation

Data Retention

The app adheres to the Salesforce data retention policies set by the customer. Data retention is managed within Salesforce.

Security & Compliance

Security: The app uses OAuth 2.0 for Google authentication. OAuth tokens are stored securely within Salesforce using Named Credentials.

Compliance: Form Today complies with:

  • Google API Services User Data Policy
  • OAuth 2.0 security standards
  • Salesforce’s security and access control standards
